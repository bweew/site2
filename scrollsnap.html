How to paginate a single page

using this css trick you can make it seem like you're scrolling through multiple pages when its actually just a single page. To get this effect heres what you do...

  1. style the html tag with scroll-snap-type: y mandatory.
  2. make a div with sections inside for each page.
  3. style the section with "height: 100vh; scroll-snap-align:start; scroll-snap-stop:normal;"
